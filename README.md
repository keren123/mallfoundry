## 许可（License）




所有的业务组件都可以嵌入到 Java 程序中，或者作为服务器、集群、云中的服务运行。它们可以作为独立的服务运行，也可以集成在一起为电商业务提供丰富的套件。它们与 Spring 完美地结合在一起。通过使用丰富的 Java 和 REST API，它们是开发电商业务的理想基础。它们支持微服务编排或将微服务本身编写成软件。
